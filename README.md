För att köra appen:

Gå in i server mappen och kör: "npm install"
Kör sendan: "npm run server" för att starta databaden

Öppna en ny terminal:
Gå in i client mappen och kör "npm install"
Kör sedan: "npm run dev" för att starta sidan

Kort beskrivning:
Server mappen:
npm i
npm run server

Client mappen:
npm i
npm run dev

Sidan körs på:
http://localhost:5173/
